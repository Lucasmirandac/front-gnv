
# Landing Page & Dashboard Design (Melhorias)

Este projeto é uma tentativa de melhoria do template original de Landing Page & Dashboard Design, adaptado para o contexto do **Galo na Veia**, programa oficial de sócio-torcedor do **Clube Atlético Mineiro**.  

O **Galo na Veia** é o sistema que centraliza os planos de associação, benefícios e vantagens para torcedores do Atlético, permitindo acesso facilitado à compra de ingressos, prioridade em jogos, descontos em produtos oficiais e experiências exclusivas. A proposta deste projeto é explorar uma interface moderna para a landing page e o dashboard desse tipo de sistema, focando em:

- Apresentar de forma clara os diferentes planos de sócio-torcedor e seus benefícios.
- Facilitar o acompanhamento do status do sócio (plano atual, pontuação, histórico de uso).
- Melhorar a experiência de navegação em dispositivos desktop e mobile.

O design base utilizado como referência está disponível em `https://www.figma.com/design/uBH5HYY75BA7IWrNIicyfJ/Landing-Page---Dashboard-Design`.

## Executando o projeto

- Instale as dependências com `npm i`.
- Inicie o servidor de desenvolvimento com `npm run dev`.
